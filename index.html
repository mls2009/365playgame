How to Play Sudoku

The Grid

Sudoku is played on a 9×9 grid, divided into 3×3 boxes. Some cells already contain numbers, known as "givens".

The Rules

Fill in the grid so that every row, column, and 3×3 box contains the digits 1 through 9 exactly once.

Strategy

Use logic to figure out which numbers go where. No guessing is needed to solve a proper Sudoku puzzle.

Tips for Beginners

  • Start with easier puzzles to learn the basic techniques.
  • Look for rows, columns, or boxes that are nearly complete.
  • Use the process of elimination to determine possible values for each cell.
  • Try the "pencil mark" technique to note possible values for empty cells.
